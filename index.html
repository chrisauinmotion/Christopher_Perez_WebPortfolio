RepliCode

Today, there are many learning resources available for those wishing to teach themselves how to code. One of the learning mediums that both beginners and experienced developers alike often seek out are video coding tutorials. Video tutorials are great learning resources but they lack the level of interactiveness that learning developers yearn for, so together with a team of developers I developed a solution called Replicode, a content creation platform that allows developers to easily create interactive coding tutorials.

MyDigiDiet

MyDigiDiet is a desktop application that helps give users insight into their digital habits. The application was created using Electron (a framework for creating native applications with web technologies like JavaScript, HTML, and CSS), React (a JavaScript library for building user interfaces), Reactstrap (a module that makes it easy to use React Bootstrap 4 components), Webpack (a module whose main purpose is to bundle JavaScript files) and React-chartjs (a module that provides rich interactive react charting components using chart.js).
